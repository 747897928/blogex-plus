主要特色 响应式 / 前后端分离 / 开箱即用
技术栈 Spring全家桶 / Vue / Layui
Gitee https://gitee.com/guangxikejidaxue/blogex
作者心语

本博客前端后台是基于layuimini开源框架实现的，前端前台首页基于Java开源项目Solo的nijigen皮肤的样式， 其余样式基于layui修修补补，前端页面渲染上基于vue， 加上jquery操作dom完成数据的渲染(虽然前端开发规范不推荐， 而且也是很老的技术，但是对我一个后端开发友好)。 样式控制选用layui(虽然layui官网已经关闭),后端基于Spring Cloud Alibaba， SpringSecurity，Mybatis-Plus等， 内置邮件回复功能，前端用了很多开源插件， 比如layuimini、tinymce、layui、 pixi-live2d-display、APlayer、prism、 OwO表情包插件、editormd、 introjs等，感谢开源社区的作者们，谢谢你们！
